What's inside

Smart Detection

Auto-detect finds sprites on irregular sheets. Grid slicer for uniform layouts. Flood-fill overlap fixer. Auto-trim empty space in bulk. Auto-tag sprites by dominant color.

Advanced Workflows

Categorize sprites with tags. Organize sprites with groups and subgroups. Bulk rename sorites with variable patterns.

Animate

Build and preview animation sequences from subcategories. Adjustable FPS, base layers, per-frame position nudging.

Create Pixel Art

Pencil, eraser, fill, eyedropper, dither, shapes, and recolor. Layer system with opacity, merge, flatten. Frame animation with onion skinning.

Projects & Tabs

Multi-tab workspaces. Export individual sprites, project zips, composed sheets, or JSON metadata. Import projects and external images back in anytime. Locql auto-save recovers work between sessions or after accidental refreshes.

Works Everywhere

Optimized for both desktop and touch devices. Fully browser based & funcations locally.
